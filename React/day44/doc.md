# Cơ chế Client-Side Rendering (CSR) và Server-Side Rendering (SSR)

# CSR: Server chỉ trả về HTML cơ bản, div root, JS, và CSS cơ bản (ví dụ: Shopee)

# SSR: Server trả về trực tiếp HTML và CSS (ví dụ: TGDĐ)

# React là một thư viện JavaScript giúp xây dựng giao diện người dùng (UI) trên Client-Side

# Next.js có thể kết hợp cả CSR và SSR

# Cơ chế render của React

- > DOM ảo (Virtual DOM) -> React DOM -> DOM thật
  > Là một đối tượng xử lý DOM trong React

# Flow render của React

- JSX -> JavaScript Compiler -> React Element -> React DOM -> HTML
